Network Analysis Triad

Clarifying the relationship between Wireshark, tcpdump, and tshark. A common misconception is that tcpdump is the CLI for Wireshark.

Wireshark

The GUI tool for deep packet inspection. User-friendly but heavy.

tcpdump

Standalone CLI packet capture tool. Essential for headless servers. Not part of Wireshark.

tshark

The official CLI for Wireshark. Supports Wireshark dissectors and filters.

Firewall Evolution: iptables vs. nftables

Why modern systems are moving to nftables for performance and simplicity.

Key Improvements

  • ‚úì Atomic Updates: Update all rules at once. No broken intermediate states.
  • ‚úì Dual Stack: One rule handles both IPv4 and IPv6 (`inet` family).
  • ‚úì Performance: Uses Sets & Maps for O(1) lookups instead of linear O(n).
  • ‚úì Syntax: Human-readable, structured like C/tcpdump.

# Migration Command

$ iptables-save > rules.txt

$ iptables-restore-translate -f rules.txt

Conceptual Performance Scaling

As rule count increases, iptables slows down linearly. nftables remains nearly constant.
